# Camper Cafe Menu

A simple café menu webpage built using HTML and CSS. This project demonstrates fundamental front end development concepts including page structure, layout styling, typography, and content organization.

## Project Overview

The Camper Cafe Menu is a static webpage that displays a café menu with different coffee drinks and desserts along with their prices. The goal of the project is to practice structuring webpages using semantic HTML and styling them using CSS.

The design mimics a physical café menu, featuring a centered menu card, themed background imagery, organized menu sections, and aligned item pricing.

## Features

- Structured HTML layout using semantic elements
- Styled café menu with coffee and dessert sections
- Clean typography and menu formatting
- Centered responsive layout
- Background image styling
- Interactive link styling using hover and active states

## Technologies Used

- HTML5
- CSS3
